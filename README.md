# java-terminal-calculator

Hello welcome to my java terminal calculator application. I made this because I wanted to learn some java coding and build some simple. 
This code of course uses java and has one main switch case styled form that calls the helper methods which do the real arithmetic. 

This also uses the scanner class built into java for user input. 

To run this all you need to do is go to releases and download the latest .exe file! This is completely opensource and simple no building required. 

Everything runs in a terminal window when you open it so all you heed to do is follow the on-screen instructions from there!
